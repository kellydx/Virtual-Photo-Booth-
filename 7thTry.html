Accessing Webcam Demo With HTML5

This is a demonstration of how to access the local camera from an HTML5 Web page, using the new navigator.getUserMedia(), and apply filters on image using CSS3 Filter Property. At this point, because of security issues, getUserMedia() should only work on https: protocol. You'll need a SSL certificate for this API to work.
